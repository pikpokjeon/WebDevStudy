# 데이터 호출 시점

1. 컴포넌트 라이프 사이클 훅
- created() : 컴포넌트가 생성되자말자 호출되는 로직

1. 라우터 네비게이션 가드
- 뷰 라우터로 특정 URL에 접근할 때 해당 URL을 막는 방법.
- 예:) 사용자의 인증 정보가 없으면 특정페이지 접근 불가
```javascript
  router.beforeEach( (to, from, next)=> () )
  // to : 이동할 url
  // from : 현재 url
  // next : to에서 지정한 url로 이동하기 위해 꼭 호출해야하는 함수
```
